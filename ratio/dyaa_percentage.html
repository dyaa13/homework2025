Part 1: What is a Percentage?

A percentage means “per hundred”.
p% means p out of 100, so p% = p/100.
Example: 25% = 25/100 = 1/4.

Part 2: Convert Between % , Fractions, and Decimals

% → decimal: divide by 100.  Example: 35% = 0.35
decimal → %: multiply by 100.  Example: 0.72 = 72%
% → fraction: write p/100 then simplify.  Example: 40% = 40/100 = 2/5
fraction → %: convert to a decimal (or equivalent /100) then ×100.
Handy conversions: 1/2 = 0.5 = 50%, 1/4 = 0.25 = 25%, 3/4 = 0.75 = 75%, 2/5 = 0.4 = 40%.

Part 3: Operations with % , Fractions, and Decimals

Convert first (usually to decimals or fractions), then calculate.
Examples:
25% + 3/5 = 0.25 + 0.60 = 0.85
3/4 × 40% = 0.75 × 0.40 = 0.30
Round to 2 decimal places when needed.

Part 4: Find a Percentage of a Number

Use: p% of N = (p/100) × N

Part 5: Given a Percentage, Find the Number

If p% of N is A:
(p/100) × N = A → N = A ÷ (p/100)

Part 6: Find the Percentage

Percentage = (part ÷ whole) × 100%

Part 7: Increase and Decrease Problems

Discount: New = Original × (1 − p/100)
Increase: New = Original × (1 + p/100)

Part 8: Applications (Real-life • Mixed • Multi-step)

Translate the story into equations, then solve step-by-step.
For overall percent: find totals first, then (part ÷ whole) × 100%.
For multi-step changes: apply each change one at a time.
