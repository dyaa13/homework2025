Equal-Height Area Method (Simple)

Key rule: If two triangles share the same height to the same line, their areas are proportional to their bases on that line.

So: Area₁ : Area₂ = Base₁ : Base₂ (when heights are equal).

Useful idea: In many diagrams, a point splits a base into parts (e.g., BD:DC). Use that ratio to split areas.

Answers in this quiz are integers (cm²). Enter only the number (no units).

Example (same height ⇒ area ratio = base ratio)

If BD:DC = 3:1, then Area(ABD):Area(ADC) = 3:1.
