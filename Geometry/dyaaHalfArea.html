What is the Half Model?

The Half Model is a quick way to find areas when a shape is split into two equal parts. If two regions have the same base and the same height, then they have the same area (even if they look slanted).

  • Key idea If a region is half of a shape, then Whole = 2 × Half.
  • Triangles Area = (base × height) ÷ 2.
  • Rectangles Diagonal splits a rectangle into two equal triangles.

Opposite-sum Model (Rectangle / Parallelogram)

If you join an interior point to the four corners of a rectangle (or any parallelogram), it creates four triangles. A powerful fact is:

Top + Bottom = Left + Right (both equal half of the whole area).

So if you know three triangle areas, you can find the fourth using: Blue = (Yellow + Green) − Red (matching the diagram style in this quiz).
